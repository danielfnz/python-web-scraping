# python-web-scraping

Laboratório de web-scraping utilizando python.

## Instalação

Utilize o [pip](https://pip.pypa.io/en/stable/) para instalar as bibliotecas.

```bash
pip install requests
pip install pandas
pip install BeautifulSoup
pip install re
pip install firebase_admin
```
